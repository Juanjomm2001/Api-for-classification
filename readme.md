## Probar localmente
Para probar que todo funciona antes de subirlo a PythonAnywhere:

Instala las dependencias:
Copypip install -r requirements.txt

Ejecuta la aplicación:
Copypython app.py

Abre en tu navegador: http://localhost:5000
